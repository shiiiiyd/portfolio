# document

Vercel Next.js。

这个文章组合使用**Next.js** 和 [Nextra](https://nextra.vercel.app/)库构建，允许写下Markdown直接展示在web界面，也可以自行美化和扩展web界面，部署和备份相当简单。

你也可以花几分钟[**部署自己的**](https://vercel.com/new/git/external?repository-url=https://github.com/vercel-solutions/nextjs-portfolio-starter&project-name=portfolio&repository-name=portfolio)博客。

你也可以在这个web项目的Github中创建Markdown文档，随后他会展示在帖子中。

**Vercel 在自动building时如果md文件中存在`<<` 或者 `>>`会building失败，正如该页面所显示需要使用符号 `` 才会building成功**
